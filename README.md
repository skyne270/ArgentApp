Argent Banking App

Overview
Argent is a JavaFX-based banking application that allows users to create accounts, log in, and perform financial transactions.

Features
- User registration and login
- Automatic account number generation
- Deposit and withdrawal
- Money transfer between users
- Transaction history tracking
- Data persistence using file serialization
- Custom UI with background and icons

Technologies Used
- Java
- JavaFX
- Object-Oriented Programming
- File Handling (Serialization)

Resources
- `icon.png` → Application window icon
- `icon2.png` → Background image used in UI

How to Run
1. Open the project in IntelliJ IDEA or any Java IDE
2. Ensure JavaFX is properly configured
3. Make sure `icon.png` and `icon2.png` are in the correct resource path
4. Run `ArgentApp.java`

Project Structure
- `ArgentApp.java` → Main application logic
- `User` class → Stores user data
- `Transaction` class → Stores transaction records
- `users.dat` → Saved user data (auto-generated)
- `icon.png`, `icon2.png` → UI resources
